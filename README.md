# Clima-App-Simples

<table>
<tr>
<td>
Um aplicativo web simples para verificar a previsão do tempo em tempo real de qualquer cidade.
</td>
</tr>
</table>


## Funcionalidades
- Busca por cidade
- Exibe temperatura e umidade atual

## Tecnologias/Linguagens
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
